# VantageData 产品功能特点及使用说明

## 产品概述

VantageData 是一款基于人工智能的数据分析平台，采用现代化的桌面应用架构（Go + Wails 2.0 + React + TypeScript + Tailwind CSS），为用户提供智能化的数据探索、分析和可视化能力。

---

## 核心功能特点

### 1. 智能数据分析引擎

#### AI 驱动的数据分析
- **自然语言交互**：通过对话方式与数据进行交互，无需编写复杂的 SQL 或代码
- **智能意图理解**：系统自动理解用户的分析意图，提供精准的分析建议
- **多轮对话支持**：支持上下文连续对话，逐步深入分析数据
- **自动图表生成**：根据分析结果自动生成适合的可视化图表

#### 多模型支持
- **Claude (Anthropic)**：支持最新的 Claude 系列模型
- **OpenAI GPT**：支持 GPT-4、GPT-3.5 等模型
- **通义千问 (Qwen)**：阿里云大语言模型
- **智谱 GLM**：智谱 AI 大语言模型
- **DeepSeek**：深度求索大语言模型
- **MiniMax**：MiniMax 大语言模型

### 2. 多源数据接入

#### 本地文件导入
- **Excel 文件**：支持 .xlsx 和 .xls 格式，自动识别多工作表
- **CSV 文件**：支持单文件或目录批量导入
- **JSON 文件**：支持结构化 JSON 数据导入

#### 数据库连接
- **MySQL**：支持 MySQL 数据库直连
- **PostgreSQL**：支持 PostgreSQL 数据库
- **SQLite**：本地 SQLite 数据库
- **Snowflake**：企业级云数据仓库集成
- **BigQuery**：Google Cloud BigQuery 数据仓库

#### 电商平台集成
- **Shopify**：OAuth 授权连接，自动同步店铺数据
- **BigCommerce**：API 集成，获取订单、产品、客户数据
- **eBay**：支持订单、财务、分析数据同步
- **Etsy**：店铺信息、商品、订单、评价数据

#### 项目管理工具
- **Jira**：支持 Cloud 和 Server 版本，同步项目、问题、工时、冲刺数据

### 3. 数据可视化

#### 图表类型
- **折线图**：趋势分析、时间序列展示
- **柱状图**：分类对比、数量统计
- **饼图**：占比分析、构成展示
- **散点图**：相关性分析、分布展示
- **热力图**：密度分析、矩阵展示
- **组合图表**：多维度数据综合展示

#### 交互功能
- **数据钻取**：点击图表深入查看详细数据
- **动态筛选**：实时过滤和筛选数据
- **图表导出**：支持图片格式导出

### 4. 分析技能系统 (Skills)

#### 内置分析技能
- **群组分析 (Cohort Analysis)**
  - 用户留存率分析
  - 按时间段（日/周/月）群组划分
  - 留存热力图生成
  - 高价值用户群组识别

- **销售漏斗分析 (Sales Funnel)**
  - 多阶段转化率分析
  - 流失点识别
  - 转化时长分析
  - 渠道效果对比

#### 技能管理
- 启用/禁用技能
- 安装自定义技能（ZIP 包）
- 技能分类管理
- 分析保护机制（分析进行中禁止修改技能）

### 5. 报告导出

#### 支持格式
- **PDF**：专业报告布局，支持中文字体
- **Excel (.xlsx)**：多工作表、自动列宽、表头样式
- **PowerPoint (.pptx)**：专业幻灯片布局，自动分页排版
- **Word (.docx)**：支持 Markdown 格式解析
- **CSV**：通用数据交换格式
- **SQL**：数据库导出脚本
- **JSON**：结构化数据导出

#### 导出到数据库
- 支持直接导出到 MySQL 数据库

### 6. 扩展能力

#### MCP 协议支持
- 支持 Model Context Protocol (MCP)
- 可连接外部 MCP 服务扩展功能

#### UAPI 集成
- 结构化数据搜索
- 多数据源支持（社交媒体、游戏、图片等）
- 标准化 JSON 响应格式

#### 搜索 API 集成
- 支持配置外部搜索引擎
- 网页内容获取和分析

### 7. 国际化支持

- **双语界面**：完整支持英文和简体中文
- **实时切换**：无需重启即可切换语言
- **参数化翻译**：支持动态内容插入
- **前后端统一**：界面和服务消息全面本地化

## 使用说明

### 快速开始

#### 1. 启动应用
双击应用图标启动 VantageData，首次启动需要选择授权模式：
- **商业模式**：输入序列号激活，使用内置 LLM 服务
- **开源模式**：配置自己的 LLM API 密钥

#### 2. 添加数据源

**方式一：导入本地文件**
1. 点击左侧面板的「添加数据源」按钮
2. 选择文件类型（Excel/CSV/JSON）
3. 选择文件并输入数据源名称
4. 点击「导入」完成

**方式二：连接数据库**
1. 点击「添加数据源」
2. 选择数据库类型
3. 填写连接信息（主机、端口、用户名、密码等）
4. 测试连接后点击「导入」

**方式三：连接电商平台**
1. 选择电商平台（Shopify/BigCommerce/eBay/Etsy）
2. 按照引导完成 OAuth 授权或填写 API 凭证
3. 系统自动同步店铺数据

#### 3. 开始分析

1. 在左侧选择要分析的数据源
2. 在底部聊天框输入分析问题，例如：
   - "分析销售趋势"
   - "哪个产品销量最高？"
   - "按月统计订单数量"
   - "用户留存率是多少？"
3. AI 将自动分析数据并生成结果

#### 4. 查看结果

- **数据表格**：在右侧面板查看详细数据
- **可视化图表**：自动生成的图表展示在对话区域
- **分析洞察**：AI 提供的分析结论和建议

#### 5. 导出报告

1. 点击导出按钮
2. 选择导出格式（PDF/Excel/PPT/Word）
3. 选择保存位置
4. 完成导出

### 高级功能

#### 使用分析技能
1. 打开设置 → Skills 管理
2. 启用需要的分析技能
3. 在对话中使用相关分析，例如：
   - "进行群组分析"
   - "分析销售漏斗转化率"

#### 配置 LLM 模型
1. 打开设置 → LLM 配置
2. 选择模型提供商
3. 填写 API 密钥和端点
4. 测试连接确认配置正确

#### 数据源管理
- **重命名**：右键数据源选择重命名
- **删除**：右键选择删除数据源
- **刷新**：电商数据源支持增量刷新
- **表管理**：可删除或重命名表和列

#### 会话管理
- **新建会话**：为不同分析任务创建独立会话
- **历史记录**：查看和继续之前的分析会话
- **清除历史**：清理不需要的会话记录
