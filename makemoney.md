# 分析包销售专家指南

本指南基于 VantageData 平台的 Marketplace 系统，帮助您从零开始创建、上传、销售和交付分析包。

---

## 一、什么是分析包

分析包（Quick Analysis Pack，.qap）是 VantageData 平台的可复用数据分析模板，包含：
- 数据查询逻辑
- 分析模板
- 可视化配置
- 预设参数

**常见类型**：
- 销售漏斗分析
-  cohort 留存分析
- 库存周转分析
- 客户画像分析

---

## 二、如何构建优质分析包

### 2.1 明确目标用户

| 用户群体 | 需求特点 | 推荐定价 |
|---------|---------|---------|
| 小型电商 | 操作简单、价格敏感 | 免费或低价 |
| 中型企业 | 专业分析、需要定制 | 按次付费 |
| 大型企业 | 深度分析、长期使用 | 订阅制 |

### 2.2 分析包结构要求

```
skills/
├── skill.json          # 元数据配置
├── SKILL.md            # 说明文档
├── input_schema.json   # 输入参数定义
└── templates/          # 分析模板
```

### 2.3 核心要素

1. **清晰的名称**：一目了然，如"Shopify 销售漏斗分析"
2. **详细的描述**：说明功能、适用场景、使用方法
3. **输入参数**：明确定义需要用户提供的参数
4. **输出结果**：说明分析包会产生什么结果
5. **兼容性**：标注支持的电商平台版本

### 2.4 最佳实践

```markdown
## 示例：销售漏斗分析包

### 功能
- 分析从访问到成交的用户转化路径
- 识别转化率低的环节
- 提供优化建议

### 适用场景
- Shopify / BigCommerce 店铺
- 月订单量 > 100 的店铺

### 使用方法
1. 导入订单数据
2. 设置分析时间段
3. 运行分析
4. 查看可视化报告
```

## 三、如何上传分析包

### 3.1 上传流程

1. **在 VantageData 客户端中**
   - 打开"分析包管理"界面
   - 右键点击要分享的分析包
   - 选择"分享到市场"

2. **配置分享设置**
   - 选择分类（Shopify / BigCommerce / eBay / Etsy 或自定义）
   - 选择定价模式

3. **等待审核**（如果需要）
   - 管理员审核后自动发布

### 3.2 定价模式选择

| 模式 | 适用场景 | 示例价格 |
|-----|---------|---------|
| 免费 | 扩大影响力、吸引用户 | 0 Credits |
| 按次付费 | 高价值、低频使用 | 10 Credits/次 |
| 限时 | 短期项目、试用 | 50 Credits/30天 |
| 订阅 | 长期核心分析包 | 20 Credits/月 |

### 3.3 定价策略建议

- **引流款**：免费或 1 Credit，建立用户基数
- **利润款**：20-100 Credits/月，针对专业用户
- **组合销售**：提供套装优惠

## 四、如何销售分析包

### 4.1 优化产品页面

1. **标题优化**
   - 包含关键词：平台 + 功能 + 目标
   - 示例："Shopify 店铺复购率分析包"

2. **描述优化**
   - 使用清晰的结构（功能、适用场景、使用步骤）
   - 配合截图和演示数据

3. **定价策略**
   - 初期可设置较低价格吸引评价
   - 获得好评后逐步提价

### 4.2 推广渠道

1. **平台内推广**
   - 在用户群中分享
   - 参与平台社区讨论

2. **社交媒体**
   - LinkedIn：面向 B2B 用户
   - 博客：发布使用教程

3. **合作伙伴**
   - 与电商咨询公司合作
   - 提供定制化分析服务

### 4.3 提高销量技巧

1. **提供试用**
   - 设置限时免费或低价试用
   - 允许用户体验后再购买

2. **收集反馈**
   - 鼓励用户评价
   - 及时回复问题

3. **持续更新**
   - 根据用户反馈改进
   - 定期发布新版本

## 五、如何交付分析包

### 5.1 自动交付

通过 Marketplace 系统自动完成：
1. 用户购买后自动下载 .qap 文件
2. 使用权限信息保存到本地
3. 用户通过"加载分析技能包"导入使用

### 5.2 权限管理

| 定价模式 | 权限检查点 |
|---------|-----------|
| 免费 | 无限制 |
| 按次付费 | 每次执行扣减 1 次 |
| 限时 | 期限内无限次 |
| 订阅 | 订阅期内无限次 |

### 5.3 客户服务

1. **使用说明**
   - 在分析包中附带详细文档
   - 提供使用视频教程

2. **问题响应**
   - 预留联系方式
   - 及时解答用户疑问

3. **更新维护**
   - 兼容平台更新
   - 修复用户反馈的问题
