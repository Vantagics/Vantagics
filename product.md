# Vantagics (万策) - 智能数据分析平台

## 产品概述

Vantagics（万策）是一款基于人工智能的桌面数据分析平台。用户通过自然语言对话即可完成数据查询、分析和可视化，无需编写 SQL 或代码。

### 技术栈

- **前端**：React + TypeScript + Vite + Tailwind CSS
- **后端**：Go + Wails (桌面应用框架)
- **AI 引擎**：CloudWeave AI (Eino)
- **数据库**：SQLite (本地缓存)
- **可视化**：ECharts

### 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11、macOS 10.15+、Linux |
| 内存 | 建议 8GB 以上 |
| 网络 | 需要网络连接以使用 AI 分析功能 |

## 核心功能特点

### 1. 多数据源支持

| 类别 | 数据源 |
|------|--------|
| 文件 | Excel (.xlsx, .xls)、CSV、JSON |
| 关系型数据库 | MySQL、PostgreSQL、SQLite、Doris |
| 云数据仓库 | Snowflake、BigQuery |
| 电商平台 | Shopify、eBay、Etsy、BigCommerce |
| 项目管理 | Jira |
| 金融数据 | Bloomberg、LSEG、Alpha Vantage、IEX Cloud、SP Global、PitchBook、Quandl |

### 2. AI 智能分析

- **自然语言查询**：用户用自然语言描述分析需求，AI 自动理解意图
- **意图建议**：智能推荐可能的分析意图
- **Python 代码生成**：自动生成并执行数据分析代码
- **实时进度显示**：分析过程中显示执行进度
- **多轮对话**：支持上下文记忆，可连续追问

### 3. 数据可视化

- **可拖拽仪表板**：自由调整布局
- **多图表类型**：折线图、柱状图、饼图、散点图、热力图等
- **关键指标卡片**：快速展示核心业务指标
- **智能洞察**：自动识别数据中的关键洞察

### 4. 分析技能（Skills）

- **同期群分析**：分析用户留存和行为模式
- **销售漏斗分析**：追踪销售转化流程

### 5. 导入/导出功能

- **数据导入**：多种格式数据导入
- **报告导出**：PDF、Excel、PowerPoint、Word
- **分析技能包**：打包分析流程，可复用

### 6. Marketplace 市场

- 分享和下载分析技能包
- 积分交易系统
- 多种定价模式（免费、按次付费、订阅）

### 7. 企业级功能

- **授权模式**：商业模式 / 开源模式
- **Credits 积分系统**
- **MCP 服务集成**
- **多语言支持**：中文 / 英文
- **代理服务器支持**
