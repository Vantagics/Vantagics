# 同期群分析 Skill

## 描述
分析不同时间段用户的留存和行为模式，帮助识别用户生命周期价值和留存趋势。

## 数据要求
- **user_id**: 用户唯一标识符
- **date**: 事件发生日期
- **event** (可选): 事件类型

## 输出
1. **留存率热力图**: 显示各个同期群在不同时间段的留存率
2. **留存曲线图**: 对比不同同期群的留存趋势
3. **数据文件**:
   - `cohort_retention_details.csv`: 详细的留存数据

## 关键指标
- 月度留存率
- 同期群规模
- 留存曲线
- 最佳/最差表现同期群

## 使用场景
- 了解新用户留存情况
- 识别流失高峰期
- 对比不同时期获客质量
- 优化用户生命周期管理

## 参数
- `cohort_period`: 同期群划分周期（默认：month）
- `retention_periods`: 分析的留存周期数（默认：12个月）
