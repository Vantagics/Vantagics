# VantageData 快速开始指南

## 欢迎使用 VantageData

VantageData 是一款基于人工智能的数据分析平台，让您通过自然语言对话即可完成数据查询、分析和可视化，无需编写 SQL 或代码。

## 第一步：选择授权模式

首次启动 VantageData 时，您需要选择授权模式：

### 商业模式（推荐）

- 使用序列号激活
- 自动配置 AI 模型
- 无需手动配置 API 密钥
- 适合大多数用户

**如何激活**：
1. 输入序列号（格式：XXXX-XXXX-XXXX-XXXX）
2. 输入邮箱地址
3. 点击"激活"按钮
4. 激活成功后即可开始使用

**没有序列号？**
- 点击"申请试用"链接
- 输入邮箱地址
- 系统自动生成 30 天试用序列号

### 开源模式

- 需要自行配置 AI 模型 API 密钥
- 支持多种 AI 模型提供商
- 适合有技术背景的用户

**如何配置**：
1. 选择"开源模式"
2. 系统自动打开设置对话框
3. 在 LLM 配置页面填写 API 密钥
4. 测试连接后保存

## 第二步：导入数据源

### 本地文件导入

**Excel 文件**：
1. 点击"添加数据源"按钮
2. 选择"Excel"
3. 选择 .xlsx 或 .xls 文件
4. 输入数据源名称
5. 点击"导入"

**CSV 文件**：
1. 点击"添加数据源"按钮
2. 选择"CSV"
3. 选择单个文件或整个文件夹
4. 输入数据源名称
5. 点击"导入"

### 数据库连接

**MySQL**：
1. 点击"添加数据源"按钮
2. 选择"MySQL"
3. 填写连接信息：
   - 主机地址（如：localhost）
   - 端口（默认：3306）
   - 用户名
   - 密码
   - 数据库名
4. 点击"测试连接"
5. 测试成功后点击"导入"

**PostgreSQL**：
1. 点击"添加数据源"按钮
2. 选择"PostgreSQL"
3. 填写连接信息（端口默认：5432）
4. 测试连接后导入

### 电商平台集成

**Shopify**（推荐一键授权）：
1. 点击"添加数据源"按钮
2. 选择"Shopify"
3. 输入店铺 URL（如：mystore.myshopify.com）
4. 点击"授权"按钮
5. 在浏览器中完成 OAuth 授权
6. 自动同步数据

## 第三步：开始分析

### 创建新会话

1. 点击左侧面板的"新建会话"按钮
2. 选择要分析的数据源
3. 会话创建成功

### 提问示例

**基础查询**：
- "显示所有订单"
- "有多少个客户？"
- "最近一周的销售额是多少？"

**趋势分析**：
- "分析过去12个月的销售趋势"
- "按月统计订单数量并画折线图"
- "哪个产品的销量增长最快？"

**对比分析**：
- "对比各地区的销售额"
- "不同产品类别的销量对比"
- "本月和上月的销售额对比"

**深度分析**：
- "分析客户购买行为"
- "找出高价值客户"
- "预测下个月的销售额"

### 查看结果

- **对话区域**：显示 AI 的分析结论和解释
- **右侧面板**：显示数据表格和图表
- **仪表盘**：展示关键指标和可视化

## 第四步：导出结果

### 导出单次分析

1. 在分析结果的仪表盘中点击"导出"按钮
2. 选择导出格式：
   - PDF：专业报告
   - Excel：数据表格
   - PowerPoint：演示文稿
   - Word：文档报告
   - CSV：纯数据
3. 选择保存位置
4. 点击"保存"

### 导出快捷分析包

1. 右键点击会话
2. 选择"导出快捷分析包"
3. 输入创作人名称
4. （可选）设置口令密码
5. 选择保存位置
6. 点击"保存"

**快捷分析包的用途**：
- 在其他数据源上重现相同的分析
- 分享给团队成员
- 上传到市场供他人使用

## 第五步：使用市场功能

### 浏览市场

1. 点击菜单栏的"市场"→"分析包管理"
2. 系统自动使用 SN + Email 登录（无需手动操作）
3. 浏览市场中的分析包
4. 按分类筛选（Shopify、BigCommerce、eBay、Etsy 等）

### 下载分析包

**免费分析包**：
1. 点击"下载"按钮
2. 直接下载到本地

**收费分析包**：
1. 查看定价信息（按次/限时/订阅）
2. 确认 Credits 余额充足
3. 点击"购买并下载"
4. 自动扣除 Credits 并下载

### 使用分析包

1. 右键点击数据源
2. 选择"加载快捷分析包"
3. 选择已下载的 .qap 文件
4. 系统验证数据源兼容性
5. 确认后自动执行分析
6. 在仪表盘中查看结果

### 分享分析包

1. 在分析包管理界面右键点击分析包
2. 选择"分享到市场"
3. 选择分类
4. 选择定价模式：
   - 免费：无需设置价格
   - 按次付费：输入每次使用的价格
   - 限时：输入价格和有效天数
   - 订阅：输入价格和计费周期
5. 点击"上传"
6. 上传成功后在市场中可见

## 常见问题

### Q: 分析结果不准确怎么办？

**A**: 尝试以下方法：
1. 更具体地描述分析需求
2. 在设置中开启"自动意图理解"
3. 增加"最大分析步骤数"
4. 检查数据源中的数据质量

### Q: 如何更换 AI 模型？

**A**: 
- **商业模式**：AI 模型由服务器配置，无法自行更换
- **开源模式**：设置 → LLM 配置 → 修改提供商和 API Key

### Q: Credits 不足怎么办？

**A**: 
- 商业模式：联系管理员充值
- 市场 Credits：通过支付方式购买

### Q: 如何在不同数据源上重复分析？

**A**: 使用快捷分析包功能：
1. 在原数据源上完成分析
2. 导出为 .qap 文件
3. 在目标数据源上导入执行

### Q: Python 环境配置失败怎么办？

**A**: 
1. 运行"Python 安装诊断"
2. 尝试"创建 VantageData 环境"
3. 确保已安装 Python 3.3+ 或 Anaconda

AI 会自动记住对话上下文：

- **短期记忆**：最近 5 条完整消息
- **中期记忆**：较早对话的摘要

**查看记忆**：
- 菜单 → 查看 Agent 记忆状态

**配置搜索 API**：
1. 设置 → 搜索 API 配置
2. 选择搜索引擎（DuckDuckGo 免费可用）
3. 填写 API Key（如需要）
4. 测试连接后保存
